### 什么是 Prompt Engineering？

**Prompt Engineering = 提示工程**

通过精心设计提示词（Prompt），让大模型输出更准确、更有用的内容。

---

### 1. Prompt 的基本原则

#### ✅ 好的 Prompt 应该包含：

| 要素 | 说明 | 示例 |
|------|------|------|
| **角色设定** | 告诉 AI 它是谁 | "你是一位资深 Java 开发者" |
| **任务描述** | 清晰说明要做什么 | "请帮我优化这段代码" |
| **输入内容** | 提供具体素材 | "以下是需要优化的代码：" |
| **输出格式** | 指定期望格式 | "请以表格形式输出" |
| **示例** | 提供参考示例 | "例如：..." |

---

#### ❌ 避免的问题：

| 问题 | 示例 | 改进 |
|------|------|------|
| **太模糊** | "帮我写个函数" | "帮我写一个冒泡排序函数" |
| **缺少上下文** | "这段代码有问题吗？" | "以下 Java 代码在处理大数据时效率低，请分析原因并优化" |
| **格式不明确** | "给我一些建议" | "请给出 5 条具体的代码优化建议，每条包含：问题、原因、解决方案" |

---

### 2. 常用 Prompt 模板

#### 📝 代码优化模板

```python
prompt = """
你是一位资深的 {语言} 开发者，专注于代码质量和性能优化。

{待优化代码}

请分析上述代码，并给出优化建议：

输出格式：
## 问题分析
[列出代码存在的问题]

## 优化建议
1. **问题名称**
   - 问题描述：...
   - 优化方案：...
   - 优化后代码：...

2. **问题名称**
   ...
"""

# 使用示例
prompt = prompt.format(
    language="Java",
    待优化代码="""
public List<User> getUsersByIds(List<Long> ids) {
    List<User> result = new ArrayList<>();
    for (Long id : ids) {
        User user = userRepository.findById(id).orElse(null);
        if (user != null) {
            result.add(user);
        }
    }
    return result;
}
"""
)
```

### 4. 进阶技巧：Chain of Thought（思维链）

**问题**：AI 在复杂任务上容易出错

**解决方案**：让 AI 分步骤思考

```python
# ❌ 直接提问
prompt = "计算 23 * 45 + 67 / 3"

# ✅ 思维链
prompt = """
计算以下表达式：23 * 45 + 67 / 3

请分步骤计算：
步骤1：计算 23 * 45
步骤2：计算 67 / 3
步骤3：将步骤1和步骤2的结果相加
步骤4：给出最终答案
"""

# 输出：
"""
步骤1：23 * 45 = 1035
步骤2：67 / 3 ≈ 22.333
步骤3：1035 + 22.333 = 1057.333
步骤4：最终答案 ≈ 1057.333
"""
```

### 机器学习核心概念（5分钟）

#### 1. 什么是机器学习？

**定义**：让计算机从数据中学习规律，而不是通过硬编码规则

**简单理解**：
```
传统编程：规则 + 数据 = 答案
机器学习：答案 + 数据 = 规则
```

**类比**：
- 教小孩子认字：
  - 传统方法：告诉孩子"横竖撇捺"的规律
  - 机器学习方法：给孩子看很多字，让他自己总结规律

---

#### 2. 机器学习三大要素

| 要素 | 说明 | 生活中的类比 |
|------|------|-------------|
| **数据** | 训练素材 | 学生的教材和练习题 |
| **算法** | 学习方法 | 学生的学习方法（背诵、理解、练习） |
| **算力** | 计算能力 | 学生的学习时间和精力 |

---

#### 3. 机器学习的主要类型

| 类型 | 标签 | 目标 | 典型应用 |
|------|------|------|---------|
| **监督学习** | ✅ 有 | 预测标签 | 图像分类、垃圾邮件检测 |
| **无监督学习** | ❌ 无 | 发现规律 | 用户分群、异常检测 |
| **强化学习** | ⭐ 奖励 | 最大化奖励 | 游戏AI、机器人控制 |

**生活类比**：
- **监督学习**：老师批改作业，告诉你对错
- **无监督学习**：自己整理房间，把相似的东西放一起
- **强化学习**：玩游戏时，根据得分调整策略
