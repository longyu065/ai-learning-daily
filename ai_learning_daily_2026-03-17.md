---

#### 📝 Prompt 三要素

| 要素 | 说明 | 示例 |
|------|------|------|
| **角色** | 告诉AI它是什么身份 | "你是一个Java开发专家" |
| **任务** | 清晰说明要做什么 | "帮我解释这段代码" |
| **要求** | 具体说明输出格式 | "用表格对比，字数500字内" |

---

#### ✅ 好的 Prompt 示例

**示例1：让AI解释代码**

```
角色：你是一个有5年经验的Java后端开发工程师
任务：帮我解释下面这段代码的作用
要求：
- 用表格形式输出（变量名、作用、说明）
- 字数控制在300字以内
代码：
public static void main(String[] args) {
    String[] names = {"Alice", "Bob", "Charlie"};
    for (String name : names) {
        System.out.println("Hello, " + name);
    }
}
```

**AI 会回答**：
```
变量名    作用          说明
names     字符串数组    存储三个名字
name      循环变量      遍历数组中的每个元素
System    输出类        控制台输出
println   方法          打印字符串
```

---

**示例2：让AI写测试代码**

```
角色：你是一个擅长单元测试的Java工程师
任务：为下面的方法写JUnit测试
要求：
- 测试正常情况
- 测试边界情况
- 用given-when-then格式注释

代码：
public int calculateScore(int score) {
    if (score < 0) return 0;
    if (score > 100) return 100;
    return score;
}
```

#### 💡 Prompt 进阶技巧（5分钟）

| 技巧 | 说明 | 示例 |
|------|------|------|
| **Few-shot** | 给几个例子 | "示例：输入1→输出A，输入2→输出B，现在输入3→?" |
| **Chain of Thought** | 让AI一步步思考 | "请一步步推理，然后给出答案" |
| **限制输出** | 控制输出格式 | "用JSON格式输出" |
| **角色扮演** | 设定人设 | "用产品经理的语气回答" |

### 🕐 第二部分：10分钟 - 机器学习基础补课

---

#### 📖 今日基础概念：什么是机器学习？

**用Java类比**：

```java
// 传统编程（您熟悉的）
public String classifyEmail(String emailContent) {
    if (emailContent.contains("中奖")) {
        return "垃圾邮件";
    } else if (emailContent.contains("发票")) {
        return "正常邮件";
    }
    // ...写几百条规则
}
```

```python
# 机器学习（新方式）
# 不写规则，给数据让模型自己学
emails = [
    ("点击领奖", "垃圾"),
    ("项目汇报", "正常"),
    ("发票下载", "正常"),
    # ...几千条数据
]

model = train(emails)  # 模型自动学习规律
```

**核心区别**：

| 传统编程 | 机器学习 |
|---------|---------|
| 您写规则（if-else） | 数据中找规律 |
| 规则固定 | 可以持续学习 |
| 适合规则明确的场景 | 适合复杂模糊的场景 |

**机器学习的本质**：
```
数据 + 机器学习算法 = 能做预测的模型

就像：
训练数据 + 经验 = 会判断的老师傅
```
