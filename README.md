## 🧮 日期计算逻辑

根据当前日期自动计算今天是第几天：

```
天数 = (当前日期 - 起始日期) % 7 + 1
```

**示例**：
- 2026-03-17 → 第1天（周一）
- 2026-03-18 → 第2天（周二）
- 2026-03-24 → 第7天（周日）
- 2026-03-25 → 第1天（周一，新周期开始）
