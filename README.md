## 🧮 学习模式：混合学习

### 时间分配

| 时间 | 内容 | 目标 |
|------|------|------|
| 🕐 20分钟 | **大模型应用** | 立即上手，保持兴趣 |
| 🕐 10分钟 | **机器学习基础** | 系统补基础，理解原理 |

### 为什么选择混合模式？

| 模式 | 优点 | 缺点 |
|------|------|------|
| **纯机器学习** | 系统性强 | 前期枯燥，3周才能接触大模型 |
| **纯应用** | 立即上手 | 理解不深，遇到问题难解决 |
| **混合模式** | ✅ 立即上手<br>✅ 逐步补基础<br>✅ 即学即用 | 需要每天坚持 |

### 学习路径图

```
第1周（应用优先）     第2-4周（基础深入）     第5-7周（项目实战）
    ↓                        ↓                      ↓
Prompt Engineering      监督学习/深度学习         RAG系统
    ↓                        ↓                      ↓
API 调用                NLP/CV/LLM             微调应用
    ↓                        ↓                      ↓
简单应用                 系统原理                 端到端项目
```

## 🧮 日期计算逻辑

根据当前日期自动计算今天是第几天：

```
天数 = (当前日期 - 起始日期) % 7 + 1
```

**示例**：
- 2026-03-17 → 第1天（周一）
- 2026-03-18 → 第2天（周二）
- 2026-03-24 → 第7天（周日）
- 2026-03-25 → 第1天（周一，新周期开始）

## 🗓️ 学习路径（7天循环）

| 天数 | 20分钟（大模型应用） | 10分钟（机器学习基础） |
|------|---------------------|---------------------|
| **第1天** | Prompt Engineering 入门 | 什么是机器学习 |
